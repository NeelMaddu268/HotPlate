Tailwind UI/UX Master Rules

Color Palette: We are building a food discovery app. Ban all default gray and purple colors.

Primary: bg-orange-500 to bg-red-500 gradients for primary actions (warm, appetizing).

Backgrounds: Use crisp bg-white or ultra-light bg-stone-50.

Text: Use text-slate-900 for headings and text-slate-500 for secondary text.

Typography: Use a modern sans-serif. Make headings bold and tight (tracking-tight).

Components (Deal Cards): 
* Borders are banned. Use soft shadows (shadow-sm resting, shadow-md on hover).

Generous padding: Use p-6 inside cards.

Images must cover their containers fully (object-cover) with slightly rounded top corners (rounded-t-2xl).

Micro-interactions: Every interactive element must have a transition (transition-all duration-200 ease-in-out). Buttons should slightly scale up on hover (hover:-translate-y-0.5).
